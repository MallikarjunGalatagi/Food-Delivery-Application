## 📖 How It Works

Foodify is a Java-based full-stack online food delivery web application that connects **Customers**, **Restaurant Owners**, **Delivery Partners**, and **Administrators** through a single platform.

Customers can register, log in securely, browse restaurants, search food items, add meals to their cart, and place orders. Restaurant owners can manage menus and process orders, delivery partners can accept deliveries and update delivery status, while administrators monitor the complete system by managing restaurants, users, and platform statistics.

The application is built using **Java, JSP, Servlets, JDBC, MySQL, HTML, CSS (Bootstrap 5), and JavaScript**, with secure session management and BCrypt password encryption.

## 📝 Registration

Users can register by entering:

- Username
- Email
- First Name
- Last Name
- Mobile Number
- Password

Additional fields based on role:

**Restaurant Owner**

- Restaurant Name
- Restaurant Address

**Delivery Partner**

- Driving License Number

All details are validated before being stored in the MySQL database.
